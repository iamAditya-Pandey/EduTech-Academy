# EduTech Academy Landing Page

Welcome to the **EduTech Academy** project! This is a premium, fully responsive, and animated landing page designed for an online education platform.

## Features
- **Modern UI/UX**: Clean layout with glassmorphism touches and premium typography.
- **Responsive Design**: Looks great on mobile, tablet, and desktop screens.
- **Scroll Animations**: Smooth entrance animations using Intersection Observer.
- **Dynamic Counters**: Animated statistics section.
- **Dark Mode Accents**: High contrast elements to draw attention to CTAs.

## Technologies Used
- HTML5 (Semantic Structure)
- CSS3 (Variables, Flexbox, Grid, Animations)
- Vanilla JavaScript (DOM manipulation, scroll events)

## Getting Started
To view this project locally:
1. Clone this repository.
2. Open `index.html` in your web browser.
3. Alternatively, serve the files using a local development server like Live Server (VS Code) or Python's `http.server`.
